マピオンベクター（ベータ）
==========
ベクターデータを活用した地図ページです。ベータ版のため以下ご了承ください。

- 情報が少ない
- 情報の取捨選択が不十分
- 地物の重なりの正確性が不十分
- 地図情報の更新が遅い場合がある

主な機能
==========
- 回転
  - マウス右ボタン（副ボタン）を左右にドラッグすることにより、地図が回転します
- バードビュー
  - マウス右ボタン（副ボタン）を上下にドラッグすることにより、地図が傾きます
- でか字
  - でか字にチェックを入れることにより、文字とアイコンが大きくなります
- キョリ測
  - キョリ測にチェックを入れて地図面をクリックすることにより、距離を測定することができます
- 有料道路
  - 有料道路にチェックを入れることにより、有料道路を表示/非表示を切り替えることができます
- 標準地図
  - 全てベクターデータで作成された地図です
- 衛星地図
  - 衛星画像の上にベクターデータの情報を乗せた地図です


対応ブラウザ
==========
WebGLに対応しているブラウザであれば問題なく動作します（WebGLを有効にしている必要があります）
